Typora: Extremely Concise Editor

2020-04-09

1 Characteristics Intro

1.1 What you see is what you get

  • Only one window.
  • Live Preview —— Just press "Enter", you will get what you write.

1.2 Convenient to edit table, code and formula —— block-level element

  • Insert table

    • Keyboard shortcuts: Ctrl + T

    • Examples

  • Insert code

    • Keyboard shortcut: type ``` and then press "Enter"

    • Examples

      def test():
    pass

  • Insert formula

2 Markdown tutorial

2.1 Paragraph

  • Edit your words directly

  • Begin a new line: two spaces + "Enter"
    Examples
    RUNOOB.COM
    GOOGLE.COM

2.2 Fonts

  • Italic Text: *Italic Text*
  • Italic Text: _Italic Text_
  • Bold Text: **Bold Text**
  • Bold Text: __Bold Text__
  • Bold and Italic Text: ***bold and italic text***

2.3 Separator Line

***

* * *

******

- - -

2.4 Delete Line

  • Formation

    ~~AB~~

  • Examples

    AB

2.5 Underline

  • Formation

    <u>text to be underlined</u>

  • Examples
    underlined text

2.6 Foot Notation

Foot Notation is the supplement of text.

  • Formation
    [^text to be noted]

  • Examples
    Create a footnote like this:[1]

2.7 Link

[link name](link address)
This is a link runoob

or

<http://www.runoob.com>
http://www.runoob.com

2.8 Advanced Link

  • Variates can be substituted for links and the variable address is appended to the end of the document.

  • Examples
    Use 1 as the variate for this link: [Google][1]
    Google
    Use 2 as the variate for this link: [Runoob][2]
    Runoob

    At the end of the document, set value(website address) for each variate.
    [1]: http://www.google.com/
    [2]: http://www.runoob.com/

2.10 Tables

  • formation

    |  head 1  |  head 2  |
|  ------  |  ------  |
|  table cell  | table cell |
| table cell  | table cell |

    | head 1 | head 2 |
    | ------ | ------ |
    | table cell | table cell |
    | table cell | table cell |

  • Set the alignment of the table

    • -: Set headings and content to right

    • :- Set headings and content to left

    • :-: Set headings and content to the center

    • Examples

      | left alignment | right alignment | center alignment |
| :-----| ----: | :----: |
| table cell | table cell | table cell |
      left alignment right alignment center alignment
      table cell table cell table cell

2.11 Advanced Skills

  • 支持的 HTML 元素

    不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。

    目前支持的 HTML 元素有: <b> <kbd> <i> <em> <sup> <sub> <br> 等 ，如：

    使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑

    使用 Ctrl+Alt+Del 重启电脑

  • 转义

    Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符：

    **文本加粗** 
\*\* 正常显示星号 \*\*

    文本加粗
    ** 正常显示星号 **

    Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号:

    \   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号

  • 公式

    当你需要在编辑器中插入数学公式时，可以使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现。提交后，问答和文章页会根据需要加载 Mathjax 对数学公式进行渲染。如：

    $$
\mathbf{V}_1 \times \mathbf{V}_2 =  \begin{vmatrix} 
\mathbf{i} & \mathbf{j} & \mathbf{k} \\
\frac{\partial X}{\partial u} &  \frac{\partial Y}{\partial u} & 0 \\
\frac{\partial X}{\partial v} &  \frac{\partial Y}{\partial v} & 0 \\
\end{vmatrix}
${$tep1}{\style{visibility:hidden}{(x+1)(x+1)}}
